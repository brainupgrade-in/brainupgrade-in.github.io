Privacy Policy

Effective Date: April 11, 2026 Last Updated: April 11, 2026 Extension: AgentGrow — AI Browser Extension Publisher: Gheware Technologies

📋 Short version: AgentGrow stores your data only on your device, encrypted. It sends your messages only to the LLM endpoint you configure. It collects zero analytics, zero telemetry, and has no servers of its own.

1. Who We Are

AgentGrow is developed and maintained by Gheware Technologies ("we", "our", "us"), a technology training and tooling company based in India. Contact: contact@gheware.com

2. Information AgentGrow Collects

AgentGrow collects and stores the following data locally on your device only:

  • Conversation history: The messages you send and receive within AgentGrow. Stored in chrome.storage.local, which is accessible only to the AgentGrow extension on your device.
  • Provider configurations: The API endpoint URL, provider name, and selected model you configure. Your API key is stored encrypted (AES-GCM-256) and is never readable in plaintext after entry.
  • Prompt templates: Any templates you save for reuse.
  • User preferences: Theme, default context mode, and keyboard shortcuts. Non-sensitive preferences are synced via your Google account's chrome.storage.sync.
  • Network audit log: Metadata about API calls (endpoint, timestamp, token count). Message content is NOT logged.

3. Information AgentGrow Does NOT Collect

AgentGrow has no servers, no backend, and no analytics infrastructure. The following data is never collected:

  • Your browsing history or the URLs you visit
  • The content of your conversations (beyond local storage on your device)
  • Personal identifiable information of any kind
  • Crash reports or error logs sent to external services
  • Usage analytics, feature tracking, or session data
  • Your IP address or device fingerprint

4. How Information Is Used

Data stored locally is used solely to:

  • Display your conversation history within the extension
  • Authenticate requests to your configured LLM provider
  • Remember your UI preferences
  • Provide the audit log feature

5. Data Sharing

AgentGrow shares data with one party only: the LLM API endpoint you configure in Settings. When you send a message, that message (and optionally the current page text, if you have enabled context mode) is sent to your configured endpoint.

⚠️ Important: The privacy practices of your chosen LLM provider (OpenRouter, OpenAI, Groq, your company's internal server, etc.) are governed by that provider's own privacy policy. AgentGrow has no control over, and no visibility into, what that provider does with the data you send. Please review your provider's policy before use.

We do not sell, share, rent, or disclose your data to any other third party.

6. Data Security

  • API key encryption: API keys are encrypted using AES-GCM-256 via the Web Crypto API before storage. They cannot be decrypted without the per-device encryption key, which is also stored locally.
  • HTTPS enforcement: AgentGrow refuses to send API calls over HTTP to non-localhost endpoints.
  • Content Security Policy: The extension enforces a strict CSP that prevents execution of any remote code.
  • Isolated architecture: API keys and provider configurations are accessible only in the extension's service worker, never in content scripts or web pages.

7. Data Retention and Deletion

Your data is retained locally until you delete it. You can delete individual conversations, all conversations, all provider configurations, or all data at any time via Options → Privacy → Delete All Data.

If you uninstall the extension, all data stored in chrome.storage.local is deleted by Chrome automatically. Sync storage is removed when you disconnect your Google account from Chrome Sync.

8. Children's Privacy

AgentGrow is not directed to children under 13. We do not knowingly collect information from children. If you are under 13, please do not use this extension.

9. Your Rights (GDPR / CCPA)

Because all data is stored locally on your device and we have no servers, we cannot access your data to fulfil erasure or portability requests through our systems. However:

  • Access: All data is visible via Options → Privacy Dashboard.
  • Portability: Export all data as JSON via Options → Privacy → Export All Data.
  • Erasure: Delete all data via Options → Privacy → Delete All Data, or uninstall the extension.
  • Correction: Edit any stored data directly within the extension.

If you have questions about your rights, contact us at privacy@gheware.com.

10. Changes to This Policy

We will post any changes to this Privacy Policy on this page and update the "Last Updated" date. For material changes, we will provide a notification within the extension. Continued use of AgentGrow after changes constitutes acceptance of the updated policy.

11. Contact

For privacy-related questions or concerns:
