2. Description of Service

AgentGrow is a free, open-source Chrome browser extension that automates common browser tasks — filling forms, writing email drafts, composing messages, summarizing documents, extracting data, and navigating complex sites — using LLM providers of your choice (such as OpenRouter, OpenAI, Anthropic, Groq, Ollama, or your own server). It is developed and maintained by Gheware Technologies.

The Extension can read the content of your active browser tab and interact with form fields and text inputs on your behalf, solely when you request it through the chat interface. The source code is available under the MIT license on GitHub. The Extension is provided free of charge.
